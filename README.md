# Vocal Pitch Monitor Web Application
The project is dedicated for developing a web-based platform for real-time vocal pitch monitoring, using signal processing and machine learning to make professional vocal training more accessible through objective feedback and personalized coaching.


1. This project is recommended to run and edit using IntelliJ IDEA.
If you don't have IntelliJ IDEA download it
https://www.jetbrains.com/idea/

2. For the database, you will also need to download MySQL to successfully run the project.
https://www.mysql.com/

3. Open the file on IntelliJ IDEA by pressing Ctrl + O.
After opening the file, click the run button in the top right corner.
If you see the bottom terminal is displaying "application is running..." that means the server is up and running

4. If you have trouble starting the server, capture the error and send it to me or create issue on github at the top left corner
